# Around The U.S. (React Edition)

Este é um projeto desenvolvido com **React** e **Vite**, focado na criação de uma página interativa de galeria de fotos onde os usuários podem adicionar, remover e curtir fotos, além de editar seu perfil.

## 🚀 Tecnologias Utilizadas

- **React**: Biblioteca principal para construção da interface.
- **Vite**: Ferramenta de build rápida para o desenvolvimento.
- **CSS3**: Estilização baseada na metodologia BEM.
- **JavaScript (ES6+)**: Lógica e manipulação de estados.

## 🛠️ Funcionalidades implementadas nesta etapa

- **Componentização Total**: A interface foi dividida em componentes menores e reutilizáveis (`Card`, `Popup`, `Header`, `Main`, `Footer`, etc.).
- **Gestão de Estados (`useState`)**: Controle de abertura e fechamento de múltiplos popups (Editar Perfil, Novo Local, Editar Avatar e Confirmação de Exclusão).
- **Interatividade**:
  - Fechamento de popups através da tecla `ESC`, clique no `overlay` ou botão de fechar.
  - Renderização dinâmica da lista de cartões.
  - Popup de imagem expandida com transições suaves.
- **Estrutura de Pastas Profissional**: Organização dos componentes em pastas individuais com caminhos planos para facilitar a manutenção.
